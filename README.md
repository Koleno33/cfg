# Dotfiles for two types of computer

## Desktop
### picom
- there is transparency in the unfocused window
- tried to make Firefox pop-ups always opaque (works intermittently)
- works properly with proprietary Nvidia drivers
- smooth window switching animation in i3

### i3
- special keyboard shortcuts for taking screenshots of the entire screen and a selected area with `maim`

### nvim
- use `lualine plugin`

## Laptop
### picom
- doesn't contain any animations
- works properly with Intel graphics drivers

### i3status
- separators between blocks
- shows the sound level

### nvim
- use `vim airline plugin`
